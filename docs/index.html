Features

AI Categorization

Claude AI automatically categorizes transactions into your YNAB budget categories based on item descriptions.

Amazon Itemization

Break down Amazon orders into individual items with proper category splits for accurate budget tracking.

Statement Parsing

Convert PDF bank statements to YNAB-compatible CSV format. Supports Chase credit cards and checking accounts.

Email Automation

Forward Amazon order emails for real-time processing via Google Cloud Functions. Fully hands-off.

Reconciliation

Compare bank statements against YNAB to find discrepancies and ensure your budget matches reality.

Batch Processing

Process historical transactions in bulk using Anthropic's Batches API at 50% reduced cost.

How It Works

Two approaches: real-time email processing or batch backfill from statements.
