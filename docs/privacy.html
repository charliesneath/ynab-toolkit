Privacy Policy

Last updated: January 28, 2026

Overview

YNAB Toolkit is a personal automation tool that categorizes Amazon purchases in YNAB (You Need A Budget). This application is for private, personal use only.

Data Collection

This application accesses:

  • Gmail: To read Amazon order confirmation emails and send categorization summary emails
  • YNAB: To create and update transaction categorizations

Data Usage

  • Email data is processed in real-time to extract order information
  • No email content is stored permanently
  • Transaction data is sent to YNAB for categorization
  • A local cache may store category mappings to improve future categorizations

Data Sharing

This application does not share any data with third parties. All data remains between:

  • Your Gmail account
  • Your YNAB account
  • Google Cloud Platform (for processing)

Data Security

  • OAuth 2.0 is used for secure authentication
  • Credentials are stored in Google Cloud Secret Manager
  • No passwords are stored by this application

Contact

This is a personal project. For questions, contact the project owner.

Changes

This privacy policy may be updated occasionally. Continued use of the application constitutes acceptance of any changes.
